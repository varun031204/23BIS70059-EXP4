# State Management Demo - 23BIS70059-EXP4

A simple React app demonstrating three types of state management:

## State Management Types

1. **Local State** - Using `useState` hook
2. **Global State (Redux)** - Using Redux Toolkit
3. **Global State (Context API)** - Using React Context

## Setup

```bash
npm install
npm run dev
```

## Components

- `LocalStateComponent` - Demonstrates local state with counter
- `ReduxComponent` - Demonstrates Redux global state with counter
- `ContextComponent` - Demonstrates Context API with theme toggle
